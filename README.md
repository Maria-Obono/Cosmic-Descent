# Cosmic-Descent
Cosmic Descent is a high-octane 3D voxel racing experience. Set in a neon-drenched futuristic world, players must navigate complex environments to collect energy cubes and reach the extraction pedestal before the clock hits zero.

#The Mission
You are dropped into a vibrant, voxelized dimension where time is your greatest enemy.

* Collect: Scavenge for energy cubes scattered across the map.

* Race: Navigate the terrain using precise movement controls.

* Survive: Reach the final pedestal before the countdown expires to secure your victory.

#Key Features
* Dynamic Voxel World: A fully realized 3D environment with high-performance rendering.

* Futuristic Aesthetic: A "matte soft yellow" and neon-blue visual palette designed for maximum immersion.

* Responsive HUD: Real-time tracking of your stats, energy collection, and remaining time.

* Advanced Scoring: A dynamic system that rewards speed and efficiency.

#Run Locally

Prerequisites: Node.js
1. Install dependencies: npm install
2. Set the GEMINI_API_KEY in .env.local to your Gemini API key
3. Run the app: npm run dev
